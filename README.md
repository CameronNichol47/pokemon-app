# Pokemon Search App

A simple web app that allows users to search for Pokémon and view their stats using the PokeAPI.

## Features
- Search Pokémon by name
- Displays sprite image
- Shows height, abilities, and game indices
- Styled with a retro pixel Pokémon UI

## Tech Stack
- JavaScript (async/await)
- HTML/CSS
- PokeAPI

## What I Learned
- Fetching data from APIs using async/await
- Working with JSON data
- Using map() to transform data
- Dynamically updating the DOM
- Basic UI styling and layout with CSS

## How to Run
1. Open `pokemon.html`
2. Enter a Pokémon name
3. Click search
